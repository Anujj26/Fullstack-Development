# Shivam Juice Center – Webpage Project

A simple and visually structured webpage created using HTML and CSS to represent a juice shop. This project focuses on layout design, styling, and practical implementation of HTML text formatting tags.

## Project Overview

This project is designed to simulate a real-world juice shop website. It combines structured content with basic styling techniques to create an organized and readable webpage.

### Objectives
- Design a multi-section webpage
- Apply CSS styling such as colors, fonts, and spacing
- Use HTML formatting tags in practical content
- Display structured data using tables

### Technologies Used
- HTML5
- CSS3

## Key Features
- Structured header and footer
- Section-based content layout
- Menu table with pricing
- Demonstration of multiple HTML formatting tags
- Consistent styling and readable design

## HTML Tags Used
- **Text Emphasis:** `<b>`, `<strong>`, `<i>`, `<em>`, `<u>`
- **Text Styling:** `<mark>`, `<small>`, `<del>`, `<ins>`
- **Scientific Formatting:** `<sub>`, `<sup>`
- **Code and Layout:** `<code>`, `<pre>`, `<br>`
- **Quotations:** `<q>`, `<blockquote>`
- **Miscellaneous:** `<abbr>`, `<cite>`

## Output Description

The webpage includes:
- An introduction section for the juice center
- A menu displaying juice items and prices
- A section explaining features of the shop
- Examples of HTML formatting tags displayed in a table

## Learning Outcomes
- Understanding HTML page structure
- Applying CSS for styling and layout
- Using formatting tags effectively in real content
- Organizing information in a structured format

## Conclusion

This project demonstrates how basic HTML and CSS can be used together to build a clean and functional webpage. It provides a strong foundation for further web development concepts.
