# Juice Shop Data Visualization Dashboard

This project is a data visualization dashboard built for a Juice Shop to track business metrics. It uses **HTML**, **CSS**, and **JavaScript** to create an interactive interface, and integrates **Chart.js** to render dynamic charts.

## Features
- **Business Overview:** Tracks total revenue and top-selling products.
- **Dynamic Charts (Chart.js):**
  - **Line Chart:** Displays the daily sales trend over the last 7 days.
  - **Bar Chart:** Shows product-wise sales to easily compare individual drink performance.
  - **Pie Chart:** Categorizes the products and displays a proportional breakdown of category distribution.
- **Interactive UI:** A refreshing design featuring a clean sidebar layout, dynamic updates, and custom icons.

## File Structure
- `index.html`: The main structural layout of the dashboard.
- `style.css`: Contains the styling for a premium visual aesthetic (utilizes predefined colors and modern fonts).
- `script.js`: Handles data initialization, DOM manipulation, and Chart.js configuration.

## How to Run it Locally
1. Simply clone this repository to your local machine.
2. Open the `index.html` file in a traditional web browser.
3. No build tools are required.
